# Format Foundry Demo Storyboard

Target runtime: 2 minutes 45 seconds. Hard stop: 2 minutes 55 seconds.

## 0:00-0:12 - Hook

Visual: `exports/youtube-thumbnail-1920x1080.png`, then the app opening.

Narration: "File work is rarely one task. Conversion, compression, metadata, archives, media, and
downloads usually force people across a pile of disconnected tools. Format Foundry brings that
work into one dependable desktop workspace."

## 0:12-0:42 - Core workflow

Visual: Add two safe sample files in Convert, choose a target, adjust one quality control, and show
the queue and output location. Keep the cursor movement deliberate.

Narration: "The workflow stays consistent: add files, choose the operation, review the controls,
then run the queue. Status, pause, stop, and output remain visible without blocking the workspace."

## 0:42-1:10 - Breadth without clutter

Visual: Move across Documents, Images, Audio, Video, Archives, and Metadata. Pause briefly on two
real controls rather than rapidly clicking every tab.

Narration: "Each tab keeps only the controls relevant to that job, while a shared shell makes the
next step predictable. The app scales and scrolls for common Windows and Linux screen sizes."

## 1:10-1:38 - Optional backend center

Visual: Open Check Updates, then the Optional Feature Tools center. Show detected status, feature
explanations, official links, and install routing. Do not install software during the recording.

Narration: "A clean consumer computer does not need Codex, Python, or preinstalled backends. The
updater detects optional tools, explains exactly what they unlock, and routes setup through trusted
package managers or official pages."

## 1:38-2:02 - Aria2 and safety

Visual: Open Aria2 Downloads and Torrents. Show the disclaimer, file selection, search, select-all,
per-file progress, pause, and stop controls. Do not download copyrighted material.

Narration: "The aria2 workspace supports ordinary URLs, magnets, Metalink, and torrent file
selection while keeping warnings, selected files, progress, pause, and stop controls explicit."

## 2:02-2:25 - Cross-platform delivery

Visual: Show the GitHub release assets or the release gallery slide. Highlight the versioned EXE,
DEB, AppImage, checksum file, and provenance attestation.

Narration: "Format Foundry ships like a consumer product: a signed Windows installer, a Debian
package for Ubuntu, a portable AppImage, uninstall paths, versioned assets, checksums, and public
build provenance."

Optional alternate visual: briefly show the opt-in Idea Bank and PC Health Snapshot workspaces before
the release slide. Describe Idea Bank only as local and disabled by default. Describe PC Health only
as private, read-only, and informational; do not imply cloud sync, collaboration, or antivirus.
